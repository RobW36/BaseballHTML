# baseballHTML
Simple button click baseball game with HTML and Javascript

# Getting Started
All code and image files are in the public_html folder.  The application contains one Javascript file for the logic and one HTML file for the UI.  Images are contained in the image folder.

# Installing
Pull down the public_html folder and open the diceball.html file in any browser.  As long as the diceball.html file is in the same directory as the diceballJS.js file and the images folder, the application will when opened in a browser.
